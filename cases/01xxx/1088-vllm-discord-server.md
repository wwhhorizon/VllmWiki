### Issue 正文摘录

As the vLLM's user and developer community keeps growing, the current communication model (Github Issues/PRs + random emails) turns out to be not efficient enough. Especially, this prevents communication across different developers without getting through @WoosukKwon and me. Therefore, we finally created the [vLLM Discord server](https://discord.gg/jz7wjKhh6g) to enable more efficient communication. Please join and participate in the discussion! Your issues can get more speedy feedback over there. We are relatively new to Discord. Please let us know if you have any suggestions.
