### 🚀 The feature, motivation and pitch ## What is Quantized Model Reloading and Why is it Useful? vLLM serves not only as a inference runtime for serving requests from end users, but also as a means of serving requests for large language model post-training. One particularly important use case is using vLLM to serve rollouts (required by RL pipelines) using a quantized model to serve the requests. For more information, see [QeRL: Beyond Efficiency – Quantization-enhanced Reinforcement Learning for LLMs](https://arxiv.org/html/2510.11696v1). However, this feature is not fully composable with all vLLM loading features.
